# ☕ Java Programming Study
## 📘 5장 : 상속

### 상속(inheritance)
- 객체 지향의 상속
  - 부모클래스에 만들어진 필드, 메소드를 자식클래스가 물려받음
  - 부모의 생물학적 특성을 물려받는 유전과 유사
- 상속을 통해 간결한 자식 클래스 작성
  - 동일한 특성을 재정의할 필요가 없어 자식 클래스가 간결해짐

### 상속의 편리한 사례
